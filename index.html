Analista / Desenvolvedor

Trabalho a mais de 15 anos na área de desenvolvimento de sistema criando software para integrar sistemas existentes e complementando funcionalidades a estes sistemas.


Tenho experiencia na integração de sistemas na área de custo e orçamento, dando apoio e executando orçamentos, acompanhamento de custo, gerando informação para tomada de decisão estratégica.


Nestas integrações de sistemas, utilizei tecnologias com Asp.Net Webforms com C# / JQuery / bootstrap, utilizando diversos bancos de dados dependendo da situação (MS Sqlserver / MySql / Oracle) criando webservices/API para integração e desenvolvimento de aplicação para adicionar, complementar ou substituir uma funcionalidade destes.


Desenvolvi aplicações mobile que substituía a importação de dados via Excel, dando agilidade, rapidez e lançamentos de dados em tempo real, proporcionando a tomada de decisão estratégica mais rápida.


Tenho me qualificado em diversas áreas do desenvolvimento, principalmente em tecnologias de Front-End, tais como frameworks javascript (Vue JS, Angular, React) onde entendia ser meu ponto fraco, além de aprofundar no DotNet como na tecnologia “core”.
