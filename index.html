Analista / Desenvolvedor

Sou Analista/Desenvolvedor com mais de 15 anos de carreira, tenho experiência na integração de sistemas legados e complementando funcionalidades a estes.


Trabalhei na área de desenvolvimento de sistema criando software para setor de orçamento e custo durante 10 anos. Atualmente trabalho desenvolvendo aplicações para setor de logística.


Durante o período que atuei para área de custo, apoiei a execução de orçamentos, gerando informação para tomada de decisão estratégica da empresa, na área de logística desenvolvo e atualizo o sistema de gerenciamento dos armazéns automatizados aplicando melhorias nas aplicações e migrando processos legados e atuo como líder técnico da equipe.


Durante minha experiência profissional, utilizei tecnologias com Asp.Net Webforms C# / Asp.Net MVC C# / Asp.Net Core C# / .Net 5 C# / JQuery / Bootstrap / Angular / JavaScript / TypeScript, utilizando diversos bancos de dados (MS Sqlserver / MySql / Oracle / Postgress), criando webservices/API e/ou RabbitMQ para integração e desenvolvendo melhorias as aplicações, adicionando, complementando ou substituindo funcionalidades.


Desenvolvi aplicações mobile (Java M.E.) que substituía a importação de dados via Excel, dando agilidade, rapidez e lançamentos de dados em tempo real, proporcionando a tomada de decisão estratégica mais rápida.


Tenho me qualificado em diversas áreas do desenvolvimento, principalmente em tecnologias de Front-End, como frameworks JavaScript/TypeScript (Vue JS, React JS).


Outras cursos que conclui e/ou estou cursando em áreas do desenvolvimento: linguagem de programação (Python e PHP), banco Firebase e MongoDb, desenvolvimento de jogos (Unity com C#) , mobile Xamarim e Flutter, React Native.
